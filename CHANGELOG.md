# Changelog

本文档记录 `blue_battery` 的主要变更。

## [0.1.0] - 2026-03-29

### Added

- WinUI 3 `MSIX` 项目骨架
- 托盘图标、左右键交互、快捷菜单
- 单实例约束与激活重定向
- 蓝牙最小闭环
- `DeviceWatcher` 自动刷新
- GATT 通知订阅与轮询回退
- 托盘最低电量聚合图标
- 状态持久化、缓存恢复与断开态保留
- 无标题栏紧凑弹出面板
- 设置界面与开机自启动
- 自签名安装证书与手动安装说明

### Changed

- 统一面板圆角并精简说明信息
- 修复打包清单能力顺序并完成正式回归验证
- 收敛应用编排并抽离设备显示/状态文案服务
- 精简 README 中的对外 TODO 与安装说明
- 统一发布版本到 `0.1.0`

### Fixed

- 修复状态持久化在修剪场景下的 JSON 警告
- 修复 `Release MSIX` 打包所需的发布配置
- 修复 `BluetoothBatteryTelemetryService.Dispose()` 的并发安全问题
